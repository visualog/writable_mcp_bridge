# Agent Recipes

These recipes are the shortest safe paths for agents working with Xbridge.

Use them before inspecting server internals:

- `start-server.md`: bring up the local bridge and verify the build.
- `inspect-current-file.md`: identify the active Figma file/session.
- `inspect-pages.md`: discover pages in the currently open file.
- `inspect-node-details.md`: read precise node/component/instance details.
- `ws-first-read.md`: use WebSocket-first reads with HTTP fallback.
- `streaming-validation.md`: validate SSE/WS/fallback health.
- `recovery-server-down.md`: recover from server-down or stale-process states.
