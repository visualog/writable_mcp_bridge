# Troubleshooting

Use these notes before opening server internals.

- `stale-server-process.md`: `/health` works but fields/version are old.
- `session-state.md`: active, stale, offline, or multiple app sessions.
- `detail-api-fallback.md`: detail APIs return `metadata_fallback`.
- `search-nodes-selection-required.md`: search requires selection or target context.
- `figma-beta-vs-figma-app.md`: Figma and Figma Beta share one local server but different sessions.
